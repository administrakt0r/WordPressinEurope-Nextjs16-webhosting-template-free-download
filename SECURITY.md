# Security Policy

## Supported Versions

Use the latest version of the application to ensure you have the most up-to-date security patches.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of our systems seriously. If you believe you have found a security vulnerability, please report it to us as described below.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please review our security contact information at:
[https://wpineu.com/.well-known/security.txt](https://wpineu.com/.well-known/security.txt)

You can also email us directly at support@wpineu.com.

## Security Procedures

1.  **Reporting**: Once we receive your report, our security team will investigate the issue.
2.  **Assessment**: We will assess the severity and impact of the vulnerability.
3.  **Fix**: If confirmed, we will work on a fix and release it as soon as possible.
4.  **Disclosure**: We will coordinate public disclosure with you once the vulnerability has been patched.

Thank you for helping keep WPinEU secure.
