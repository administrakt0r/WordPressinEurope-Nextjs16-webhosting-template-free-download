# Security Policy

## Supported Versions

The following versions of the WPinEU project are currently being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

We take the security of our systems seriously. If you believe you have found a security vulnerability, please report it to us as described below.

### How to Report

Please report security vulnerabilities by emailing [support@wpineu.com](mailto:support@wpineu.com).

You should expect to receive a response within 24-48 hours.

### Policy

*   Please do not publicly disclose the vulnerability until we have had a chance to fix it.
*   We will keep you updated on our progress.

For more details, please refer to our security.txt at [https://wpineu.com/.well-known/security.txt](https://wpineu.com/.well-known/security.txt).
